# View-DB - Просмотр базы данных

Тестовая программа для просмотра и управления данными в базе данных.

## Использование

1. Откройте в браузере: `http://localhost:3000/view-db`
2. Выберите тип БД:
   - **Локальная БД** - использует `DATABASE_URL` из `.env`
   - **Рабочая БД** - использует `DATABASE_URL_PRODUCTION` из `.env` (если настроено)
3. Выберите таблицу из списка и нажмите "Открыть"
4. Просматривайте данные с пагинацией
5. Используйте CRUD операции:
   - **Создать** - кнопка "+ Создать" вверху страницы
   - **Изменить** - кнопка "Изменить" в строке таблицы
   - **Удалить** - кнопка "Удалить" в строке таблицы

## Настройка переменных окружения

Для работы с разными БД добавьте в `.env`:

```env
# Локальная БД (используется по умолчанию)
DATABASE_URL="postgresql://..."

# Рабочая БД (опционально)
DATABASE_URL_PRODUCTION="postgresql://..."
```

Если `DATABASE_URL_PRODUCTION` не указан, будет использоваться `DATABASE_URL` для обеих опций.

## Доступные таблицы

- `users` - Пользователи
- `notes` - Заметки
- `categories` - Категории
- `prompts` - Промты
- `tags` - Метки
- `votes` - Голоса

## Особенности

- Пагинация: 10 записей на страницу
- Автоматическая сортировка по `createdAt` (если есть) или по `id`
- Поддержка всех CRUD операций
- Валидация данных на стороне сервера
